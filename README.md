# 🎬 Movie Recommender System

A content-based Movie Recommender System built with **Python**, **Machine Learning**, and **Streamlit**. The application recommends movies similar to a selected title using cosine similarity and displays movie posters fetched from the TMDB API.

## 🚀 Live Demo

🔗 **Streamlit App:** https://movie-recommender-system-y6duk5rxhhfbyficqpux4t.streamlit.app/



---

## 📌 Features

- 🎥 Recommend similar movies instantly
- 🖼️ Displays movie posters using the TMDB API
- ⚡ Fast recommendations using precomputed similarity scores
- 🌐 Interactive web interface built with Streamlit
- 📱 Simple and responsive UI

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/aditiii2/movie-recommender-system.git
```

### Navigate to the project

```bash
cd movie-recommender-system
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```

---

## 🎯 How It Works

1. Select a movie from the dropdown menu.
2. Click **Recommend**.
3. The system calculates similar movies using cosine similarity.
4. Posters are fetched from the TMDB API.
5. Five recommended movies are displayed.
